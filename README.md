# Eternalvoidco

## Projekt leírás
Ez a repository az Eternalvoidco weboldal forráskódját tartalmazza.

## Fő tulajdonságok
- Az oldal **Vercelen van hostolva**.
- A weboldalon lévő **Shop gomb össze van kötve egy Shopify webshoppal**.
- A kódverziók GitHubon vannak kezelve.

## Környezet
- Hosting: Vercel
- Forráskód tárhely: GitHub
- Webshop integráció: Shopify

## Működés röviden
A felhasználó a Shop gombra kattintva a Shopify webshop felületére jut, ahol a vásárlás történik.
